# FINVISE — Website Prototype

A clickable prototype of the new Finvise website. Double-click index.html to view.

## Pages
- index.html — Home
- lenders.html — For lenders
- insurers.html — For insurers
- public-bodies.html — For public bodies
- valuation-firms.html — For valuation firms (incl. Partnerships section)
- about.html — About us (+ management team, compliance)
- cases.html — Cases
- contact.html — Contact

## Brand & positioning
- Logo: circle "F" + FINVISE wordmark (assets/finvise-logo.svg)
- Indigo #0a1059 · Newsreader (serif) + Inter (sans) · light-only · responsive
- New generation valuation firm — never "AI"/"tech". Technology = "proprietary tools built in-house".
- B2C app appears only as a footer link.
